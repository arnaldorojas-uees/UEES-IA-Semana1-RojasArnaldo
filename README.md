# Laboratorio Semana 1 - IA

Este laboratorio incluye ejercicios prácticos en Google Colab sobre:

- Fundamentos de NumPy y Pandas
- Visualización de datos con Matplotlib, Seaborn y Plotly
- Modelos básicos de Machine Learning con Scikit-learn
- Introducción a Deep Learning con TensorFlow/Keras

## Notebooks

1. `01_Fundamentos_NumPy_Pandas.ipynb`: Arrays con NumPy y análisis exploratorio con Pandas.
2. `02_Visualizacion_Datos.ipynb`: Gráficos estáticos e interactivos.
3. `03_Machine_Learning_Basico.ipynb`: Clasificación y regresión con Scikit-learn.
4. `04_Deep_Learning_Intro.ipynb`: Red neuronal simple para clasificación de imágenes.

## Requisitos

Instalar las librerías indicadas en `requirements.txt` o usar Google Colab.
